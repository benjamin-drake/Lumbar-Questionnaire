## Spinal Clinic Patient Questionnaire

A self-contained, offline-capable HTML tool for use in NHS spinal outpatient waiting areas. Patients complete an Oswestry Disability Index (ODI v2.1), red flag screen, comorbidities and medications checklist, body pain diagram, and VAS pain scores on a tablet before their consultation. No data is stored or transmitted — the form resets after each patient. Output is a single-page printable PDF summary formatted for clinical review and EPR scanning.

Built for South Tyneside and Sunderland NHS Foundation Trust Spinal Outpatient Clinic. Single HTML file, no dependencies, no internet connection required.
